# chat2goal Copy-Paste Prompts

These prompts are for chats where you cannot install skills, plugins, custom GPTs, or project-level tools.

Use them by copying one full prompt into a fresh chat, then paste the transcript or project chat bundle where the prompt asks for input.

| Prompt | Use when |
|---|---|
| `chat2goal.prompt.md` | One chat transcript should become a generic executable goal-loop prompt. |
| `project-chat2goal.prompt.md` | Many chats from one project should become one project-level goal prompt. |
| `chat2lazycodex.prompt.md` | One chat transcript should become a LazyCodex/OMO-ready prompt. |
| `project-chat2lazycodex.prompt.md` | Many project chats should become a LazyCodex/OMO-ready plan or execution prompt. |

These are not installable skills. They are plain prompts for trying the workflow anywhere a model accepts a long paste.
